Formal Software Verification

  • Logical correctness of code gets proved mathematically
  • Formal verification tools are used to check the code
  • This process is time-consuming and requires expertise

Our innovation

  • SparkBright consists of two modules: LLM and verification
  • LLM makes suggestions for code and proof snippets
  • Verification module checks the suggestions for logical correctness
  • The two modules interact through an intelligent feedback loop
  • The developer only sees suggestions that are logically correct
